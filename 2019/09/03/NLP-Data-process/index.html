Abstract

For any NLP or machine learning application, data or dataset is the most important part. In this section, I am gonna to introduce what data looks like, what attributes data has and how to scrap data from websites.

Data Attributes

Categorical Data Attributes

Categorical data is also named as qualitative data. The features of categorical data are as follows:

  • Categories are more descriptive
  • Categorical data might overlap with each other
  • Categorical data is divided into two groups: ordinal data and Nominal

Ordinal data

This type of data attribute is a metric for non-numeric concepts such as satisfaction level, happiness level, discomfort level, and so on.

For instance:

  • Question: how would you rate our hotel service?
  • Options:
    • Bad
    • Average
    • Above average
    • Good
    • Great

All the aforementioned options will measure how you are satisfied with the hotel. Obviously, it is difficult to convert your answer to a numeric value because answers will vary from person to person. In the other hand, if a person says Good and another person says Above average, there may be a chance that they both feel the same about the hotel service. That is a reason why people can not precisely decide the numerical values for categorical data.

Nominal data

Nominal data is also named as quantitative data or numeric data, which also contains two groups: continuous data and discrete data.

This kind of data attribute is used to record data that does not overlap. Numeric data represents measurable quantity. For example, gender (Male or Female), height, weight, and so on.
