Edit Distance

Definition

Edit distance is a metric that compares words and other strings. It measures how similar two strings are based on the number of edits it takes to change one string into other.

Minimum Edit Distance

Minimum edit distance between two strings is defined as the minimum number of editing operations (operations like insertion, deletion, substitution) needed to transform one string into another.

Levenshtein Distance

The Levenshtein distance between two sequences is the smalleset number of operations of converting one string to another. There are two version of Levenshtein distance.
